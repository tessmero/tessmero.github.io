My "no-bake cookie" browser javascript game engine.

Games built with this engine can run in the web browser directly from raw source code. Or, they can be compiled and developed with sophisticated tools like [ESLint](https://eslint.org/) and [Mocha](https://mochajs.org/).

I'm still working on this privately and it has a long way to go before I would recommend anyone else try to use it. But you can access the last open source version [here](https://github.com/tessmero/idle.git)
